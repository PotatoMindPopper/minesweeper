## Usage

Here's how to use this file:

    - [-] Means not started
    - [/] Means work in progress
    - [+] Means done
    - [ ] Means nothing, just a new line

In markdown, it looks like this:

```markdown
␣␣␣␣- [-] Means not started
␣␣␣␣- [/] Means work in progress
␣␣␣␣- [+] Means done
␣␣␣␣- [ ] Means nothing, just a new line
```
